출력 조건:
- 반드시 JSON 스키마에 맞출 것
- 기록된 근거 범위 안에서만 작성할 것
- 복약 순응도 향상, 안전성, 생활 관리, 위험 신호 안내를 균형 있게 포함할 것

- 보호자용 요약은 실제 관리 행동으로 옮길 수 있게 작성할 것
- 서론형 인삿말 없이 바로 환자별 복약 핵심부터 작성할 것
- "필요 시 복용" 메모가 있는 약은 정시 복용처럼 쓰지 말 것
- 같은 정보를 content_text와 sections에 기계적으로 반복하지 말 것
- content_text는 자연스럽게 읽히는 요약본, sections는 화면 표시용 핵심 정리로 구분할 것
- 보호자용 요약은 "오늘 바로 확인할 행동" 위주로 짧고 선명하게 작성할 것
