# 에이아이 헬스 문서

이 문서는 `AI_Health_final` 저장소의 **현재 코드 상태(2026-03-19 기준)**를 기준으로 작성되었습니다.

핵심 목표는 다음 3가지입니다.

1. 팀 공통 기준으로 기능/요구사항/API를 빠르게 이해한다.
2. 배포와 운영 시 필요한 설정/절차를 누락 없이 점검한다.
3. 요구사항 반영 상태를 코드 근거와 함께 추적한다.

## 문서 바로가기

- 서비스 개요: `서비스 목적, 사용자, 범위, 목표`
- 요구사항 정의서: `기능/비기능 요구사항`
- 요구사항 반영 추적표: `REQ ID 기준 구현/테스트 근거`
- 기능 동작 시나리오: `초대코드, OCR, 복약체크, 모드전환`
- API 명세: `엔드포인트 요약`
- 배포/운영 가이드: `로컬/운영 배포, 체크리스트`
- 테스트/품질 관리: `현재 테스트 체계와 실행법`
- 이슈/리스크: `현재 기준 개선 필요 항목`

- Worker: `ai_worker/` (OCR/AI 비동기 작업)
