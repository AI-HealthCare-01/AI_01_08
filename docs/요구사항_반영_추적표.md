1. AUTH 고도화: 인증(이메일/휴대폰), 약관동의, 로그인 에러코드 분리
2. DRUG 안전성: 최대투여량 판정(REQ-DRUG-012), DUR 레벨링(011/013) 고도화
3. EXT 미구현: 수동 복약스케줄 CRUD(REQ-EXT-001/002), 이미지 분석 014~016
4. NFR 운영성: SLO 측정, 구조화 로그/trace ID, 포트 노출 정책 정리
