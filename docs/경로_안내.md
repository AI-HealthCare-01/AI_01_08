# 경로 안내

## 1) 웹 라우트 (FastAPI에서 정적 페이지 서빙)

`app/main.py` 기준 주요 경로:

- 로그인/회원가입: `/auth-demo/login`, `/auth-demo/signup`
- 앱 엔트리: `/auth-demo/app`
- 앱 하위:
- `/auth-demo/app/dashboard`
- `/auth-demo/app/health-profile`
- `/auth-demo/app/documents`
- `/auth-demo/app/caregiver`
- `/auth-demo/app/ai`
- `/auth-demo/app/drug-search`
- `/auth-demo/app/schedule`
- `/auth-demo/app/medication-check`
- `/auth-demo/app/settings`
- `/auth-demo/app/profile`

정적 자산:

- `/auth-demo/app/assets/*`
- `/auth-demo/app/mascot.png`

## 2) 프론트 페이지 매핑

`frontend/src/App.jsx` 및 `frontend/src/components/AppLayout.jsx` 기준:

| 경로 | 화면 컴포넌트 | 설명 |
|---|---|---|
| `/auth-demo/app` | `App.jsx` (홈) | 역할별 홈 |
| `/auth-demo/app/dashboard` | `AdminDashboard.jsx` | 관리자 대시보드 |
| `/auth-demo/app/health-profile` | `HealthProfile.jsx` | 건강 프로필 |
| `/auth-demo/app/documents` | `DocumentManagement.jsx` | 문서/OCR |
| `/auth-demo/app/caregiver` | `NotificationPage.jsx` | 알림센터 |
| `/auth-demo/app/ai` | `AiPage.jsx` | AI 가이드/채팅 |
| `/auth-demo/app/drug-search` | `DrugSearchPage.jsx` | 약 검색 |
| `/auth-demo/app/schedule` | `SchedulePage.jsx` | 병원 일정 |
| `/auth-demo/app/medication-check` | `MedicationCheckPage.jsx` | 복약 체크 |
| `/auth-demo/app/settings` | `SettingsPage.jsx` | 설정/초대코드 |
| `/auth-demo/app/profile` | `App.jsx` 프로필 섹션 | 내 정보 |

## 3) API 라우트 그룹

`/api/v1` 하위:

- `/auth`, `/public`
- `/users` (내정보/연동/건강프로필)
- `/documents`, `/schedules`, `/calendar/hospital`
- `/notifications`, `/settings`, `/dashboard`
- `/guides`, `/chat`
